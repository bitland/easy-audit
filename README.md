# Easy Audit

This tool is designed to simplify the job of crypto-currency auditors.

## Installation

``` sh
npm install -g easy-audit
```

## Usage

``` sh
easy-audit audit path/to/liabilities.json path/to/assets.json
```

## Run tests

``` sh
npm test
```

You can also generate a code coverage report:

``` sh
npm run coverage
```
